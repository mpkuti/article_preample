article_preample
================

My LaTeX preample, to be used used with the document class IEEEtran.

main.tex:
---------

	\documentclass[10pt,final,conference,letterpaper]{IEEEtran}

	\newcommand{\myauthor}{Mika Kutila}

	\newcommand{\mysubject}{\mytitle}
